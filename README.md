# Expense Tracker (Core Java + JDBC)

## 📌 Overview

Console-based application to manage daily expenses using Core Java and MySQL.

## 🚀 Features

* Add expense (amount, category)
* View all expenses
* Monthly category-wise report
* Input validation & exception handling

## 🛠 Tech Stack

* Java (Core)
* JDBC
* MySQL

## 🗄 Database Schema

Table: `expenses`

* id (INT, PK)
* amount (DOUBLE)
* category (VARCHAR)
* date (DATE)

## ▶️ How to Run

1. Create DB: `expense_db`
2. Create table `expenses`
3. Add MySQL Connector JAR
4. Run `Main.java`
